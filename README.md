# koa-cli
Simple CLI for scaffolding Koa projects

```
$ npm install koa-cli -g
$ koa init my-project

$ cd my-project/App

# install dependencies
$ npm install

# serve with hot reload at localhost:8080
$ npm run dev

# build for production with minification
$ npm run build

# run unit tests
$ npm run unit

# run e2e tests
$ npm run e2e

# run all tests
$ npm test


$ cd my-project/Server

# install dependencies
$ npm install

# start up server
$ npm run start
```
